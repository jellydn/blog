Mailwind - Tailwind CSS to design HTML emails.


Demo: https://github.com/jellydn/mailwind-demo


## Install


```sh
npm install -g mailwind
```


## Usage


Generate the corresponding CSS file


```sh
mailwind --input-html email.html --output-css style.css
```


Generate an inlined HTML file


```sh
mailwind --input-html email.html --output-html email-inlined.html
```
