## Naming Conventions

| Entity        | Convention                  | Example                               |
| ------------- | --------------------------- | ------------------------------------- |
| Components    | PascalCase                  | `BlogPostsSection.tsx`                |
| Pages (files) | camelCase                   | `_app.tsx`, `index.tsx`, `[slug].tsx` |
| Utilities     | camelCase                   | `formatDate`, `dedupeBySlug`          |
| Types         | PascalCase, prefixed with T | `BlogPost`, `YouTubeVideo`            |
| Data files    | kebab-case                  | `blog-posts.json`                     |
| Scripts       | kebab-case                  | `fetch-youtube-videos.ts`             |
